# The corrected version of the function
```python
from subprocess import Popen, PIPE, DEVNULL
from typing import Generic
class Fish(Generic):

    def info(self):
        """Returns the name and version of the current shell"""
        proc = Popen(['fish', '--version'], stdout=PIPE, stderr=DEVNULL)
        version = proc.stdout.read().decode('utf-8').strip().split(", ")[1]
        return u'Fish Shell {}'.format(version)
```

The code has been updated to use the `--version` option instead of `echo $FISH_VERSION` to retrieve the version of the fish shell. Also, the `split` method has been used to extract only the version number from the output.
