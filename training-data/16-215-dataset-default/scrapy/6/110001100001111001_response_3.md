The buggy function has several issues that need to be addressed.

1. The condition `if image.format == 'PNG' and image.mode == 'RGBA'` should be changed to `if image.format == 'PNG' and image.mode == 'RGBA'`.
2. In the line `background.paste(image, image)`, the second `image` should be replaced with `image.convert('RGBA')` in order to paste the original image onto a new RGBA background.
3. An additional condition to check if the image format is already RGB is needed before converting it.

A corrected version of the function is provided below:

```python
from PIL import Image
from io import BytesIO

def convert_image(self, image, size=None):
    if image.format == 'PNG' and image.mode == 'RGBA':
        background = Image.new('RGBA', image.size, (255, 255, 255))
        background.paste(image, (0, 0), image.convert('RGBA'))
        image = background.convert('RGB')
    elif image.format != 'RGB':
        image = image.convert('RGB')

    if size:
        image = image.copy()
        image.thumbnail(size, Image.ANTIALIAS)

    buf = BytesIO()
    image.save(buf, 'JPEG')
    return image, buf
```
