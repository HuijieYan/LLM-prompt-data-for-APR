    ordered : bool, optional
        Whether or not this categorical is treated as an ordered

           When `dtype` is provided, neither `categories` nor `ordered`
           should be provided.

    Returns
    -------
    Categorical

    Examples
    --------
    >>> dtype = pd.CategoricalDtype(['a', 'b'], ordered=True)

    [a, b, a, b]
