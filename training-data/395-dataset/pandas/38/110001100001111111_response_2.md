The reason for the bug is that the comparison variable is incorrectly named, causing the comparison logic to be incorrect. This can lead to unexpected behavior and in this case, an `IndexError`.

            codes=rcodes + [comp_ids],
            names=rnames + ["__placeholder__"],
            verify_integrity=False
        )

    if isinstance(data, Series):

        dummy.index = dummy_index
