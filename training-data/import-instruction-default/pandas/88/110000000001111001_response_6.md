#### Buggy function analysis

The `pivot_table` function is meant to create a spreadsheet-style pivot table as a DataFrame. The bug occurs when the function recursively calls itself to process a list of aggregation functions. This recursive call results in the incorrect handling of the `values` parameter, as well as other issues related to data manipulation within the function, leading to incorrect results.

#### Bug location
The bug likely arises from the recursive call to `pivot_table` within the aggregation function list handling.

#### Bug cause
The recursion within the function creates issues in the processing of data and column keys, leading to unexpected behavior.

#### Possible approaches to fix the bug
1. Modify the function to handle aggregation function lists without recursive calls.
2. Maintain consistent handling of `values` and proper data manipulation within the function for correct results.
