### Bug Fix Analysis
The bug occurs when the pivot_table function is passed a multi-level column index. The bug is caused by the code not properly handling the case when the column index has multiple levels.

### Bug Fix Strategy
1. Handle the case when the column index has multiple levels by checking the nlevels of the table's columns before directly accessing them.
2. Make sure that the function properly handles both single and multi-level case.
