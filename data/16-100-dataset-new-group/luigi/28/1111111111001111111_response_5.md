The bug in the `table_exists` function is caused by not handling the case-insensitivity of table names properly when checking if a table exists in the Hive metadata.

The bug can be fixed by converting both the table name and the stdout to lowercase before comparing them.

```python
def table_exists(self, table, database='default', partition=None):
    if partition is None:
        stdout = run_hive_cmd('use {0}; show tables like "{1}";'.format(database, table))

        return stdout and table.lower() in stdout.lower()
    else:
        stdout = run_hive_cmd("""use %s; show partitions %s partition
                            (%s)""" % (database, table, self.partition_spec(partition)))

        if stdout:
            return True
        else:
            return False
```
