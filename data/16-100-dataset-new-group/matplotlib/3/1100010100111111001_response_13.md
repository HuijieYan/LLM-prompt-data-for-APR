## The bug
The `is_filled` method is returning `False` when the `fillstyle` is set to `'none'`, but the expected value is `True`. This indicates that the `_recache` function is not properly setting the `_filled` attribute based on the `fillstyle`.

## The fix
We need to update the `_filled` attribute based on the `fillstyle` when recaching. If the `fillstyle` is `'none'`, then `_filled` should be set to `False`, otherwise it should be set to `True`.

## The corrected version
```python
def _recache(self):
    if self._marker_function is None:
        return
    self._path = _empty_path
    self._transform = IdentityTransform()
    self._alt_path = None
    self._alt_transform = None
    self._snap_threshold = None
    self._joinstyle = 'round'
    self._capstyle = 'butt'
    if self._fillstyle == 'none':
        self._filled = False
    else:
        self._filled = True
    self._marker_function()
```
