The issue arises from the `_recache` method of the `MarkerStyle` class not correctly updating the `self._filled` attribute when the `fillstyle` is set to `'none'`. To fix this, the `_recache` method needs to be modified to ensure that the `self._filled` attribute reflects the intended fillstyle, i.e., `False` when `fillstyle` is set to `'none'`.

```python
class MarkerStyle():

    def _recache(self):
        if self._marker_function is None:
            return
        self._path = _empty_path
        self._transform = IdentityTransform()
        self._alt_path = None
        self._alt_transform = None
        self._snap_threshold = None

        self._marker_function()
```
