### Analysis
The error message from the failing test indicates that the `info` function is returning the incorrect value. It is appending the extra "fish, " to the version number when constructing the return string.

The cause of this bug is in the line `version = proc.stdout.read().decode('utf-8').strip()`. It reads the output from the process as a byte string, then decodes it to UTF-8, but it does not strip the leading "fish, " from the version number.

### Bug Fix
To fix the bug, we need to strip the leading "fish, " from the version number before constructing the return string.

### The corrected version of the function
```python
def info(self):
    """Returns the name and version of the current shell"""
    proc = Popen(['fish', '-c', 'echo $FISH_VERSION'],
                 stdout=PIPE, stderr=DEVNULL)
    version = proc.stdout.read().decode('utf-8').strip().replace("fish, ", "")
    return u'Fish Shell {}'.format(version)
```
