To fix the bug, we need to modify the way the version is extracted from the output of the Popen command and include the prefix 'Fish Shell'. Here's the corrected version of the function:

```python
class Fish(Generic):
    def info(self):
        """Returns the name and version of the current shell"""
        proc = Popen(['fish', '-c', 'echo $FISH_VERSION'],

        version = proc.stdout.read().decode('utf-8').strip()
        return u'Fish Shell ' + version
```
