The cause of the bug is that the `version` variable is being assigned the output of the `Popen` command directly without filtering out the unnecessary part. This causes the returned value to include "fish, version" which leads to the assertion error in the failing test.

# The corrected version of the info() function
def info(self):
    """Returns the name and version of the current shell"""
    proc = Popen(['fish', '-c', 'echo $FISH_VERSION'],
                 stdout=PIPE, stderr=DEVNULL)
    version = proc.stdout.read().decode('utf-8').strip()[13:]  # Extract version number
