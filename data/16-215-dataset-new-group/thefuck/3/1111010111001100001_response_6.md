The bug in the `info` function lies in the command passed to Popen to retrieve the fish version and in the decoding of the stdout. The `Popen` command should be `['fish', '--version']` instead of `['fish', '-c', 'echo $FISH_VERSION']`. Additionally, the call to `proc.stdout.read()` should be replaced with `proc.communicate()[0]` to properly fetch the output and handle the process's return code.

Here's the corrected version of the `info` function:
```python
def info(self):
    """Returns the name and version of the current shell"""
    proc = Popen(['fish', '--version'], stdout=PIPE, stderr=DEVNULL)
    version = proc.communicate()[0].decode('utf-8').strip().split(', version ')[1]
    return u'Fish Shell {}'.format(version)
```
