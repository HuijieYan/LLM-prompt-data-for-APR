## The bug
The `info` function in the `Fish` class returns the version of the Fish shell using the `echo $FISH_VERSION` command, but the test is looking for the version using the `fish --version` command. This causes the test to fail, and it also triggers a recursive loop.

## Fix strategy
To fix the bug, we need to change the command used to get the version of the Fish shell to `fish --version` instead of `echo $FISH_VERSION`.

## The corrected version of the function
```python
def info(self):
    """Returns the name and version of the current shell"""
    proc = Popen(['fish', '--version'],
                 stdout=PIPE, stderr=DEVNULL)
    version = proc.stdout.read().decode('utf-8').strip()
    return u'Fish Shell {}'.format(version)
```
