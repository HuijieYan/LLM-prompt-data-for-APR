## Bug Explanation
The issue arises because, in the buggy function, the `_get_numeric_data()` method is being used to separate the data to compute the quantile. This method removes non-numeric columns (like datetime data) and leaves the DataFrame empty, causing downstream errors.

## Bug Fix Strategy
The bug can be fixed by checking if the data contains any numeric columns. If it doesn't, the function should return the quantile directly instead of attempting to compute it on empty data.
