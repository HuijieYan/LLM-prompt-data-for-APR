## Error Location
The error occurs in the `_get_level_number` method of the `MultiIndex` class of the `pandas.core.indexes.multi` module.

## Cause of the Bug
The bug is caused by the `_unstack_multiple` function not handling hierarchical columns properly. Specifically, it fails to convert levels to numbers when getting the column locations.

## Suggested Fix Strategy
To fix the bug, ensure that the `_unstack_multiple` function properly converts level names to integers when dealing with hierarchical columns.
