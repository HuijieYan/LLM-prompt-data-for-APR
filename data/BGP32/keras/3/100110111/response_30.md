### Analysis
The issue originates in the `_clone_functional_model` function due to the handling of multi-output layers without mask support. This leads to the failure of computing the model outputs during the cloning process.

The failing test case includes a model with multiple outputs, one of which is generated by a Lambda layer lacking mask support. This situation triggers the error message when calling `clone_model`.

The function doesn't account for the scenario where the Lambda layer doesn't support masks, causing discrepancies in the computation and resulting in the reported error.

### Fix Strategy
To address the bug, we need to handle cases where layers lack mask support, particularly in multi-output situations. By ensuring that unsupported layers are handled correctly during the cloning process, we can resolve the issue reported in the GitHub thread.

### Corrected Function
```python
def _clone_functional_model(model, input_tensors=None):
    if not isinstance(model, Model):
        raise ValueError('Expected `model` argument to be a `Model` instance, got ', model)
    if isinstance(model, Sequential):
        raise ValueError('Expected `model` argument to be a functional `Model` instance, got a `Sequential` instance instead:', model)

    layer_map = {}
    tensor_map = {}
    if input_tensors is None:
        input_tensors = [Input(batch_shape=layer.batch_input_shape, dtype=layer.dtype, sparse=layer.sparse, name=layer.name) for layer in model._input_layers]
    input_layers = []
    for layer in input_tensors:
        layer_map[layer] = layer
        input_layers.append(layer)
    
    for x, y in zip(model.inputs, input_tensors):
        tensor_map[x] = (y, None)

    depth_keys = list(model._nodes_by_depth.keys())
    depth_keys.sort(reverse=True)
    for depth in depth_keys:
        nodes = model._nodes_by_depth[depth]
        for node in nodes:
            layer = node.outbound_layer

            if layer not in layer_map:
                new_layer = layer.__class__.from_config(layer.get_config())
                layer_map[layer] = new_layer
                layer = new_layer
            else:
                layer = layer_map[layer]
                if isinstance(layer, InputLayer):
                    continue

            reference_input_tensors = node.input_tensors
            reference_output_tensors = node.output_tensors

            computed_data = []
            for x in reference_input_tensors:
                if x in tensor_map:
                    computed_data.append(tensor_map[x])

            if len(computed_data) == len(reference_input_tensors):
                kwargs = node.arguments if node.arguments else {}
                computed_tensors, computed_masks = zip(*computed_data)
                output_tensors = to_list(layer(computed_tensors, **kwargs))
                if hasattr(layer, 'compute_mask'):
                    output_masks = to_list(layer.compute_mask(computed_tensors, computed_masks))
                else:
                    output_masks = [None] * len(output_tensors)

                for x, y, mask in zip(reference_output_tensors, output_tensors, output_masks):
                    tensor_map[x] = (y, mask)

    output_tensors = []
    for x in model.outputs:
        assert x in tensor_map, 'Could not compute output ' + str(x)
        tensor, _ = tensor_map[x]
        output_tensors.append(tensor)
    
    return Model(input_layers, output_tensors, name=model.name)
```

This corrected version of `_clone_functional_model` function addresses the multi-output handling and supports cases where layers lack mask support, ensuring the cloning process is executed correctly.
