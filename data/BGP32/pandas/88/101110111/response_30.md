The bug occurs when the `pivot_table` function is called with multi-index columns but no index specified. The function encounters an error when trying to access the `columns` attribute on a Series object, which doesn't have this attribute. This behavior leads to an `AttributeError` during the function execution.

## Bug Cause
The bug arises due to an assumption made in the implementation that the data structure returned by the function will always be a DataFrame. However, when aggregating over columns only, the data structure ends up being a Series in the buggy cases, thus leading to the AttributeError.
